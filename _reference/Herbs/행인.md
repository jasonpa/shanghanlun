#### 처방 분석

* 표준 용량 : 50개
* 최대용량방 : [마황탕]({{ site.formulaurl }}/마황탕)_(70개)_ "無汗而喘"
* 동명처방 : 없음
* 단순처방 : 없음

#### 관련 증상 : （胸滿）喘咳 > 短氣；結胸；心痛；形體浮腫

胸間停水

* 病位가 胸部 ... 胸滿을 동반
* 水病고 관련

`喘咳` ☞ [병증 喘]( {{site.sympurl}}/천 )

* 〔마황〕마황탕, 마행감석탕 ← → 〔복령〕목력행인감초탕
* 〔마황〕소청룡탕 ← → 〔복령〕영감강미신하인탕

形體浮腫：주로 얼굴로 나타남

#### 결론


* 〔喘咳〕胸間停水（胸滿）로 喘咳가 생겨난 경우
